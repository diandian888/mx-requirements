* 关于"该次上课该教师的上课次数"

  * 对于新教师，可能存在2个新老师同上一个班级的1次课
  * 学校对于"课次"的计算，是按照教师人数进行均分的
  * 这2个新教师，每个教师每次上课的"上课次数" = 1 / 2 = 0.5
