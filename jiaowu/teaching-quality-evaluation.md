# 教师评价

## 评价活动
* 包含内容
  * 评价标题
  * 学期
* 1个评价活动可以包含多个评价记录

## 评价记录
* 1条评价记录属于1个评价活动
* 包含内容
  * 通用内容
    * 考核项
    * 评价方法
    * 班级
    * 1个或者多个考官（可以有1个考官账号，选择除自己外的多个教师作为考官）
    * 绑定的该班级的一次课（可选）
    * 评价总分
  * 针对“学生作品评价”考核项的内容
    * 班级的一次课的1个或者多个学生作品的平均分作为评价总分

## 评价方法
* 客观子项目打分
  考核项目细分为若干个子项目，分别打分，最后按照子项目的权重，计算总分

* 主观打分
  直接对考核项目打总分，忽略子项目

## 考核项
| 序号 | 考核项 | 评价对象 | 子项目 |
| :--: | :--: | :--: | :--: |
| 1 | 学生作品评价 | 一个班级的一次上课的学生作品 | 作品子项目评分表 |
| 2 | 随堂听课评价 | 教师在一个班级的一次上课情况 | 上课情况子项目评分表 |
| 3 | 亲子活动评价 | 教师在一个班级的一次亲子活动情况 | 亲子活动子项目评分表 |
| 4 | 保留率评价（不在本文档范围） | 教师作为责任教师的班级的学生的保留率 | n/a |

* 考核项子项目
  * 包含内容
    * 内容
    * 权重

* 考核项子项目组
  * 有一个或者多个考核子项目组成 

* 创建考核项时，需要创建一个默认子项目组（包含默认子项目以及默认权重）
* 可以按照不同的课程，可以创建不同的子项目组。形成“课程” -> “子项目组“的对应关系。
* 如果评价记录中的班级的课程，可以在“课程” -> “子项目组“找到相应的“课程”，那么使用对应的“子项目组”，否则使用默认子项目组

## 客观子项目打分
* 设定1个或者多个子项目，作为一个子项目组
* 每个子项目分数：`0` - `100`
* 每个子项目有单独权重：`0%` - `100%`
* 总分 = 子项目分数 x 权重的和
* 子项目可定制
  * 每个“**评分类型** - **课程**”的组合可以定制独立的子项目组（权重）
  * 举例
    
    | 评分类型 | 课程 | 子项目组（权重） |
    | :--: | :--: | :--: |
    | 学生作品评价 | 一年级 | A(25%), B(75%) |
    | 学生作品评价 | 四年级 | A(10%), C(40%), D(50%) |
    | 亲子活动评价 | 幼中 | E(50%), F(50%) |
    | 亲子活动评价 | 三年级 | G(30%), H(20%), J(50%) | 
* 初始化子项目
  * 目前学校给出一个适用于**所有**课程的**学生作品评价**的子项目组，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 观察细致，造型精准 | 25% |
    | 构图新颖，组织合理 | 25% |
    | 配色和谐，涂色精美 | 25% |
    | 主体明确，形象创新 | 25% | 

  * 目前学校给出一个适用于**所有**课程的**随堂听课评价**的子项目组，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 教学目标设定 | 25% |
    | 教学方法过程 | 25% |
    | 教学态度语言 | 25% |
    | 教学知识技能 | 25% | 

  * 目前学校给出一个适用于**所有**课程的**亲子活动评价**的子项目组，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 教学目标设定 | 25% |
    | 教学方法过程 | 25% |
    | 教学态度语言 | 25% |
    | 教学知识技能 | 25% | 

* 分数（子项目分数/总分）与档位的关系

   | 分数范围 | 档位 |
   | :--: | :--: |
   | 0 - 59 | 不合格 |
   | 60 - 79 | 合格 |
   | 80 - 89 | 良 |
   | 90 - 100 | 优 |
  
## 统计数据

  * 查询组合条件

    | 查询条件 | 默认值 |
    | :--: | :--: |
    | 学期 | 当前学期 |
    | 校区 | 校区记录中的第1个 |
    | 考核活动 | 当前最新的考核活动 |
    | 评价类型（考核项） | 无 |
    | 评价方法 | 无 | 否 |
    | 教师 | 无 |
    | 课程 | 无 |

  * 统计数据的展示
    * 展示在查询结果的上方
    * 按照评价类型（考评项）分类展示
    * 当前评价类型（考评项）的记录数量
    * 总分的平均分
    * 总分的平均分对应的等级（档位）占比
    * 右侧显示等级占比的饼图

  * 查询结果按照评价类型（考核项）进行分组，按照总分排序（默认降序）

  * 每条记录显示的内容
 
    * 考核标题，学期，校区，课程，评价类型（考核项），评价方法，被考核人，考官，班级（课次），总分（点击查看详情），子项分数（客观子项目打分），等级, 时间
