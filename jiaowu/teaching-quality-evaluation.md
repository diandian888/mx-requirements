# 教师评价

## 评价周期
* 按照学期
  * UI展示任意维度，首先选择学期 

## 分类

| 序号 | 类型 | 评价对象 | 评价方法 | 采样内容（适用于客观子项目打分） | 分数 |
| :--: | :--: | :--: | :--: | :--: | :--: |
| 1 | 学生作品评价 | 一个班级的一次上课的学生作品 | 1. 客观子项目打分 2. 主观打分 | 学生作品（作品子项目评分表） | 采样的作品的平均分 |
| 2 | 随堂听课评价 | 教师在一个班级的一次上课情况 | 1. 客观子项目打分 2. 主观打分 | 评价人的评价(上课情况子项目评分表) | 一次上课的平均分（1个或者多个评价人） |
| 3 | 亲子活动评价 | 教师在一个班级的一次亲子活动情况 | 1. 客观子项目打分 2. 主观打分 | 评价人的评价（亲子活动子项目评分表） | 一次亲子活动的平均分（1个或者多个评价人） |
| 4 | 保留率评价（不在本文档范围） | 教师作为责任教师的班级的学生的保留率 | 保留率计算 | 保留率 | 平均保留率 |

## 评价方法

| 评价方法 | 采样次数 | 平均分获得方法 |
| :--: | :--: | :--: |
| 客观子项目打分 | n(n >= 1) | 单次采样总分 = 子项目的分数 x 权重的和，平均分 = 所有采样总分的和 / n |
| 主观打分| 0（不采样） | 平均分 = 直接打分 |

* 保留率计算不在此文档范围
* 每次评价，只能选择1种评价方法（客观子项目打分/主观打分）
* 统计分析的时候，按照不同的评分方法，分开单独统计分析
  * 流程举例：选择学期：2019年暑期学期 -> 选择评分方法：主观打分 -> 展示所有评价记录

## 客观子项目打分
* 设定1个或者多个子项目
* 每个子项目分数：`0` - `100`
* 每个子项目有单独权重：`0%` - `100%`
* 总分 = 子项目分数 x 权重的和
* 子项目可定制
  * 每个“**评分类型** - **课程**”的组合可以定制独立的子项目（权重）
  * 举例
    
    | 评分类型 | 课程 | 子项目（权重） |
    | :--: | :--: | :--: |
    | 学生作品评价 | 一年级 | A(25%), B(75%) |
    | 学生作品评价 | 四年级 | A(10%), C(40%), D(50%) |
    | 亲子活动评价 | 幼中 | E(50%), F(50%) |
    | 亲子活动评价 | 三年级 | G(30%), H(20%), J(50%) | 
* 初始化子项目
  * 目前学校给出一个适用于**所有**课程的**学生作品评价**的子项目，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 观察细致，造型精准 | 25% |
    | 构图新颖，组织合理 | 25% |
    | 配色和谐，涂色精美 | 25% |
    | 主体明确，形象创新 | 25% | 

  * 目前学校给出一个适用于**所有**课程的**随堂听课评价**的子项目，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 教学目标设定 | 25% |
    | 教学方法过程 | 25% |
    | 教学态度语言 | 25% |
    | 教学知识技能 | 25% | 

  * 目前学校给出一个适用于**所有**课程的**亲子活动评价**的子项目，之后可能会修改

    | 子项目 | 权重  |
    | :--: | :--: | 
    | 教学目标设定 | 25% |
    | 教学方法过程 | 25% |
    | 教学态度语言 | 25% |
    | 教学知识技能 | 25% | 

* 分数（子项目分数/总分）与档位的关系

   | 分数范围 | 档位 |
   | :--: | :--: |
   | 0 - 59 | 不合格 |
   | 60 - 79 | 合格 |
   | 80 - 89 | 良 |
   | 90 - 100 | 优 |

* 一次客观子项目打分，对应一个样本列表
  * 一条客观子项目打分记录，可以展开查看绑定的样本列表，对样本进行添加/编辑/删除

## 主观打分
* 一次评价没有子项目，也没有采样
* 由所有评价人协商出最后的分数，直接对总分的平均分进行打分（总分的平均分 = 手动打分）

* 统计数据

  * 查询组合条件

    | 查询条件 | 默认值 | 必选 | 备注 |
    | :--: | :--: | :--: | :--: |
    | 学期 | 当前学期 | 是 | 评价周期是1个学期 |
    | 校区 | 校区记录中的第1个 |  否 | 按校区统计 |
    | 评价类型（考核项） | 无 | 否 |  |
    | 教师 | 无 | 否 | 按教师统计 |
    | 课程 | 无 | 否 | 按课程统计 |

  * 统计数据的展示
    * 展示在查询结果的上方
    * 按照评价类型（考评项）分类展示
    * 当前评价类型（考评项）的记录数量
    * 总分的平均分
    * 总分的平均分对应的等级（档位）占比
    * 右侧显示等级占比的饼图

  * 每条记录显示的内容
 
    考核标题，学期，校区，课程，评价类型（考核项），评价方法，被考核人，考官，班级（课次），总分（点击查看详情），等级, 时间
