# 正式班开课后学生转班

## 目的
正式班开课后，学校可以新增转班时间段（批次），允许学生提交申请，转入到相同课程对应的其他班级

## 符合条件的标准
* 课程一致
* 总价一致
* 课次数量一致
* 原班级剩余未上课次 <= 新班级剩余未上课次

## 申请的处理
* 有处理权限的员工手工处理
* 分类显示可以供转班的班级（可以勾选1个或者多个分类）

  | 分类 | 条件 |
  | :--: | :--: |
  | 1 | 同时满足：申请的时间段相同，班级有空余名额 |
  | 2 | 班级有空余名额 |
  | 3 | 其他的班级 |

* 在新班级的剩余未上课次列表中，逐一勾选需要上课课次
* 勾选课次数 = 原班级剩余未上课次
