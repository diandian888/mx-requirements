# 班级的学生列表，新增“转出转入的学生”成功的记录

## 针对学生
* 正式班开课前转班申请，处理**成功**后的学生
* 正式班开课后转班申请，处理**成功**后的学生

## UI
* 在班级的学生列表展示页面，可以有菜单/按钮，点击展示该班级的所有转班**成功**的记录
  * 转入该班的记录
  * 从该班级转出的记录
